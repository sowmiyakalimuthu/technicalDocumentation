Introduction

CSS is a stylesheet language that is used to describe the presentation of a document written in HTML or XML.

Syntax

CSS consists of a selector and a declaration block. The selector determines which HTML elements the styles will be applied to, while the declaration block contains one or more declarations separated by semicolons.

selector { property: value; }

Selectors

Selectors are patterns that match elements in a document. They can be based on element names, IDs, classes, attributes, or relationships between elements.

Common selectors include:

  • Element selector (div, p, h1, etc.)
  • Class selector (.class-name)
  • ID selector (#id-name)
  • Attribute selector ([attribute=value])
  • Descendant selector (parent child)
  • Adjacent sibling selector (element + element)
  • General sibling selector (element ~ element)

Properties

CSS properties define how HTML elements are styled. Each property has a name and a value.

Common properties include:

  • color
  • font-size
  • background-color
  • margin
  • padding
  • border
  • width
  • height
  • display
  • position
  • float
  • text-align

Box Model

The box model describes the layout of elements in CSS. It consists of the content area, padding, border, and margin.

Layout

CSS layout refers to the arrangement of elements on a web page. It can be achieved using various techniques such as floats, positioning, and flexbox.

Flexbox

Flexbox is a layout model that allows elements to be aligned and distributed within a container. It provides a more efficient way to layout, align, and distribute space among items in a container.

.flex-container { display: flex; flex-direction: row; /* or column */ justify-content: space-between; /* or space-around, space-evenly, flex-start, flex-end, center */ align-items: center; /* or flex-start, flex-end, stretch, baseline */ } .flex-item { flex: 1; /* or any other flex value */ /* Other styling for flex items */ }

Grid

CSS Grid Layout is a two-dimensional layout system that allows you to create complex grid-based layouts with rows and columns. It provides a powerful and flexible way to design responsive layouts.

.grid-container { display: grid; grid-template-columns: repeat(3, 1fr); /* 3 columns with equal width */ grid-gap: 20px; /* Gap between grid items */ } .grid-item { /* Styling for grid items */ }

Media Queries

Media queries allow you to apply different styles based on the characteristics of the device or viewport, such as screen size, resolution, or orientation. They are commonly used for creating responsive designs that adapt to different devices and screen sizes.

@media screen and (max-width: 768px) { /* Styles for screens smaller than 768px */ }

Animations and Transitions

CSS animations and transitions allow you to animate the properties of an element over time. Animations provide more control and flexibility, while transitions are simpler and easier to use for basic effects.

/* Animations */ @keyframes slide-in { 0% { transform: translateX(-100%); } 100% { transform: translateX(0); } } .element { animation: slide-in 1s ease-out; } /* Transitions */ .transition-example { width: 100px; height: 100px; background-color: blue; transition: width 0.5s ease-out; } .transition-example:hover { width: 200px; }
